# Convert-degrees

Hello! This my simple convertor Celsius to Fahr or Fahr to Celsius


## How use programm
Git clone this repo
```sh
git clone https://github.com/EduardER-developer/Convert-degrees.git
```

Use your compiler C (i use gcc)
```sh
gcc temp.c -o temp
```

Run our programm
```sh
./temp
```
